# webpack-app-kits
> Webpack application kits.

## installation
```shell
npm install @feizheng/webpack-app-kits
```

## update
```shell
npm update @feizheng/webpack-app-kits
```

## basic react project
```js
plugins.provide({
  IfElse: ['react-if-else', 'default'],
  RCM: ['react-condition-manager', 'default'],
  cx: 'classnames'
})
```
