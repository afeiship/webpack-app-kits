# webpack-app-kits
> Webpack application kits.

## installation
```shell
npm install @jswork/webpack-app-kits
```

## basic react project
```js
plugins.provide({
  IfElse: ['react-if-else', 'default'],
  RCM: ['react-condition-manager', 'default'],
  cx: 'classnames'
})
```
